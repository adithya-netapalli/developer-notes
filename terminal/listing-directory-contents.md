# Listing Directory Contents

## ls Command

### Definition
The `ls` command is used to display the list of files and folders inside a directory.

---

## Syntax

```bash
ls
```

---

## Example

```bash
ls
```

### Example Output

```text
index.html
script.js
style.css
```

---

## ls -a

### Definition
The `ls -a` command displays all files including hidden files.

Files starting with `.` are hidden files in Linux.

---

## Syntax

```bash
ls -a
```

---

## Example Output

```text
.
..
.bashrc
index.html
script.js
```

---

## ls -la

### Definition
The `ls -la` command displays:
- files and folders,
- hidden files,
- permissions,
- owner details,
- file sizes.

---

## Syntax

```bash
ls -la
```

---

## Symbolic Links

A symbolic link is a shortcut that points to another file or folder.

It is similar to shortcuts in Windows.

---

## Summary

The `ls` command is used to view files and folders inside directories with different options like:
- `ls`
- `ls -a`
- `ls -la`
