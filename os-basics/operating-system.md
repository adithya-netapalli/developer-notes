# Operating System

## What is an Operating System?

An Operating System (OS) is a special software that manages hardware resources and allows applications to run on a computer.

It decides how hardware resources such as:

- CPU
- RAM
- Storage

are shared between different applications.

## Kernel

The Kernel is the core part of the Operating System.

Its main job is to:

- Manage hardware resources.
- Communicate between software and hardware.

## Single-Core Processor

A single-core processor can execute only one task at a time.

However, we can run multiple applications simultaneously because the Operating System rapidly switches between applications.

## Context Switching

The Operating System gives each application a small amount of CPU time.

It rapidly switches applications in and out of the CPU thousands of times per second.

This process is called **Context Switching**.

Because of context switching, multiple applications appear to run at the same time.

## Summary

- Operating System manages hardware resources.
- Kernel is the core part of the OS.
- A single-core CPU executes one task at a time.
- Context Switching allows multiple applications to run efficiently.
