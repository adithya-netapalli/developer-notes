# Tags vs Elements

## What is a Tag?

A Tag is the part written inside angle brackets (`<>`).

Tags are used to define HTML elements.

Example:

```html
<h1>
</h1>
```

Here:

- `<h1>` is the opening tag.
- `</h1>` is the closing tag.

## What is an Element?

An Element is the complete structure created using tags and content.

Example:

```html
<h1>Hello World</h1>
```

Here, the entire line is an HTML element.

It includes:

- Opening tag
- Content
- Closing tag

## Example

```html
<p>This is a paragraph.</p>
```

Tag:

```html
<p>
</p>
```

Element:

```html
<p>This is a paragraph.</p>
```

## Tags and Elements

When we use a tag, an element is created in the browser.

The tag is only a representation.

The element is what actually appears in the browser.

## Non-Closing Tags

Some tags do not have a closing tag.

Examples:

```html
<hr>
<img src="image.jpg" alt="Image">
```

These are called non-closing or self-closing tags.

They still create elements in the browser.

## Summary

- Tags are written inside angle brackets (`<>`).
- Tags can have opening and closing parts.
- An element is the complete structure created using tags.
- Elements are what appear in the browser.
- Some tags are non-closing tags.
