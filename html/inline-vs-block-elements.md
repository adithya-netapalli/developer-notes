# Inline and Block Elements

## Block Elements

Block elements start on a new line and take the full available width by default.

Examples:

```html
<h1>Heading</h1>
<p>Paragraph</p>
<div>Content</div>
```

Common Block Elements:

- `<h1>` to `<h6>`
- `<p>`
- `<div>`
- `<section>`
- `<article>`
- `<header>`
- `<footer>`

## Inline Elements

Inline elements do not start on a new line.

They only take the width required by their content.

Examples:

```html
<span>Hello</span>
<a href="#">Link</a>
<strong>Text</strong>
```

Common Inline Elements:

- `<span>`
- `<a>`
- `<strong>`
- `<em>`
- `<img>`

## Div Tag

The `div` tag is a block element.

It is used to group and organize content.

Example:

```html
<div>
  <h2>Title</h2>
  <p>Description</p>
</div>
```

## Span Tag

The `span` tag is an inline element.

It is used to style or group small portions of text.

Example:

```html
<p>
  Hello <span>World</span>
</p>
```

## Difference Between Div and Span

| Div | Span |
|------|------|
| Block Element | Inline Element |
| Starts on a new line | Stays on the same line |
| Takes full width | Takes only required width |
| Used for larger sections | Used for small text/content |

## Summary

- Block elements start on a new line.
- Inline elements stay on the same line.
- `div` is a block element.
- `span` is an inline element.
- Both are used for grouping content.
