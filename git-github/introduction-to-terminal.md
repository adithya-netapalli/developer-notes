# An Introduction to Terminal and CLI

## Ways to Interact with a Computer

We can interact with a computer in mainly 2 ways:

### 1. Graphical User Interface (GUI)

- Uses buttons, windows, and mouse clicks.
- Example:
  - File Explorer
  - Chrome Browser

---

### 2. Command Line Interface (CLI)

- Uses text commands to interact with the computer.
- Commands are typed inside the terminal.

---

## Common CLI Commands

### ls

Used to list files and folders inside a directory.

```bash
ls
```

---

### pwd

Used to print the current working directory.

```bash
pwd
```

---

### mkdir

Used to create a new folder.

```bash
mkdir folderName
```

---

### cd ..

Used to move one level back to the parent directory.

```bash
cd ..
```

---

### rm

Used to permanently delete files.

```bash
rm fileName
```

---

### code .

Used to open the current folder in Visual Studio Code.

```bash
code .
```

---

## Summary

CLI helps developers interact with the computer using commands instead of graphical interfaces.
